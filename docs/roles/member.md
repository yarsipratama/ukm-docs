# Anggota UKM

Anggota UKM adalah mahasiswa yang terdaftar secara resmi dalam satu atau lebih UKM.

## Hak Akses Anggota UKM

Sebagai Anggota UKM, Anda dapat:

- Melihat profil dan status keanggotaan
- Mengakses daftar event UKM
- Mendaftar atau mengikuti kegiatan UKM
- Melihat riwayat partisipasi kegiatan

## Manfaat untuk Anggota

- Informasi kegiatan selalu terpusat dan terkini
- Proses pendaftaran event lebih mudah
- Riwayat aktivitas UKM terdokumentasi dengan baik
