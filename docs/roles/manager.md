# Pengurus UKM

Pengurus UKM memiliki peran utama dalam pengelolaan organisasi dan aktivitas UKM melalui sistem ini.

## Hak Akses Pengurus UKM

Sebagai Pengurus UKM, Anda dapat:

- Mengelola data anggota UKM
- Menyetujui atau menonaktifkan status keanggotaan
- Membuat dan mengelola event UKM
- Melihat laporan keanggotaan dan kegiatan

## Alur Umum Penggunaan

1. Login ke sistem menggunakan akun pengurus
2. Mengelola data anggota melalui menu Manajemen Anggota
3. Membuat dan mengatur kegiatan melalui menu Manajemen Event
4. Memantau partisipasi dan dokumentasi kegiatan
