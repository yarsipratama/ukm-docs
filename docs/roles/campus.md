# Pihak Kampus

Pihak Kampus berperan sebagai pengawas dan evaluator kegiatan UKM di lingkungan kampus.

## Hak Akses Pihak Kampus

Sebagai Pihak Kampus, Anda dapat:

- Melihat daftar UKM dan data anggotanya
- Mengakses informasi event dan kegiatan UKM
- Memantau aktivitas UKM secara keseluruhan
- Menggunakan data sebagai bahan evaluasi dan pelaporan

> ℹ️ Pihak Kampus tidak memiliki akses untuk mengubah data UKM secara langsung.
